# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/subarray-sum-equals-k/description/)。

- 简单地说，就是给一个整数数组和整数k，统计其中和为 k 的子数组的数量。

# 思路分享
- 虽然下意识觉得可以用滑动窗口，但是本题其实并不能用，因为元素内不仅有正数，还可能有负数，导致窗口的变化并非是单调的。

- 前缀和的思路，可以参考 {% post_link 303区域和检索-数组不可变 %}

- 通过前缀和可以方便地看到一段段的差值，这些差值就可以看作是子数组的和，于是就可以转到枚举，枚举右维护左，也就是遍历前缀和的元素，记录已遍历前缀和，就能方便的看有多少个差为目标值的前缀和元素组合。
> https://leetcode.cn/problems/subarray-sum-equals-k/solutions/2781031/qian-zhui-he-ha-xi-biao-cong-liang-ci-bi-4mwr/

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        s = list(accumulate(nums, initial=0))
        rec = defaultdict(int)
        ans = 0
        for i in s:
            ans += rec[i-k]
            rec[i] += 1

        return ans
```
