# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/reverse-linked-list/description/)。

- 简单地说，就是给一个链表，现在需要把链表的元素翻转过来。

# 思路分享
- 只能说，涉及到了链表，只能沉下心慢慢模拟指针的变化，熟悉其中的原理，然后就写出来了。膜拜灵神！

- 灵神的思路很简单但也很高效，对于每两个节点，先保存第三个节点，因为翻转后这个节点就丢失了，要先保存，然后让第二个节点指向第一个节点，再往后处理就是。
> https://leetcode.cn/problems/reverse-linked-list/solutions/1992225/you-xie-cuo-liao-yi-ge-shi-pin-jiang-tou-o5zy/

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        pre = None
        cur = head
        if head is None:
            return head
        while cur:
            nxt = cur.next
            cur.next = pre
            pre = cur
            cur = nxt
        return pre

```
