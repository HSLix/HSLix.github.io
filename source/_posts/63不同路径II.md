# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/unique-paths-ii/description/)。

- 简单地说，就是给一个二维数组，其中的 0 和 1 分别代表空和障碍物，求从该数组的左上角移动到右下角的所有路径数目。

# 思路分享
- 就是在前一题的基础上加了个障碍物，只要在那个位置置零，说明不可能有路径走到那个位置就行。具体的实现思路请看前文 {% post_link 62不同路径 %}
> https://leetcode.cn/problems/unique-paths-ii/solutions/3059862/jiao-ni-yi-bu-bu-si-kao-dpcong-ji-yi-hua-451i/

class Solution:
    def uniquePathsWithObstacles(self, obstacleGrid: List[List[int]]) -> int:
        # 一维数组
        m = len(obstacleGrid)
        n = len(obstacleGrid[0])
        f = [0]*(n+1)
        f[1] = 1
        for i in range(m):
            for j in range(n):
                # print(f)
                if obstacleGrid[i][j] == 1:
                    f[j+1] = 0
                else:
                    f[j+1] = f[j] + f[j+1]
        return f[n]
