# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/minimum-falling-path-sum-ii/description/)。

- 简单地说，就是给一个二维数组，现在要从第一行走到最后一行，但是两个相邻行的取数不能同列

# 思路分享
- 此题最大的难点在于读题，是相邻行不能同列，并非只能相邻的列。之后的思路与 {% post_link 931下降路径最小和 %} 基本相同，故不赘述。

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def minFallingPathSum(self, grid: List[List[int]]) -> int:
        # 递推
        ans = 0
        n = len(grid)
        if n == 1:
            return grid[0][0]
        # f[i][j] = min(f[i+1][k] for k in range(n) if k != j) + grid[i][j]
        f = [[0]*n for _ in range(n+1)]
        for i in range(n-1, -1, -1):
            for j in range(n):
                f[i][j] = min(f[i+1][k] for k in range(n) if k != j) + grid[i][j]
        return min(f[0])

        # 记忆化搜索
        ans = 0
        n = len(grid)
        @cache
        def dfs(i:int, j:int) -> int:
            if not (0 <= j <= n-1):
                return inf

            if i == n-1:
                return grid[i][j]
            
            return min(dfs(i+1, k) for k in range(n) if k != j) + grid[i][j]
        
        return min(dfs(0, i) for i in range(n))
```
