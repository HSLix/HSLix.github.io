# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/reverse-nodes-in-k-group/description/)。

- 简单地说，就是给一个链表和一个数字 k，现在要从左到右，每 k 个就翻转链表。

# 思路分享
- 本题对于每 k 个的遍历不是难点，重点还是在局部的翻转链表，具体思路不赘述，可以看 {% post_link 92反转链表II %}
> https://leetcode.cn/problems/reverse-nodes-in-k-group/solutions/1992228/you-xie-cuo-liao-yi-ge-shi-pin-jiang-tou-plfs/

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseKGroup(self, head: Optional[ListNode], k: int) -> Optional[ListNode]:
        n = 0
        cur = head
        while cur:
            n += 1
            cur = cur.next
        
        m = n // k
        dummy = ListNode(0, head)
        p0 = dummy
        for _ in range(m):
            pre = p0
            cur = pre.next
            for i in range(k):
                nxt = cur.next
                cur.next = pre
                pre = cur
                cur = nxt
            nxt = p0.next
            p0.next.next = cur
            p0.next = pre
            p0 = nxt

        return dummy.next
```
