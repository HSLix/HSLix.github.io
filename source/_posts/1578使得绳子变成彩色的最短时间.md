# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/minimum-time-to-make-rope-colorful/?envType=daily-question&envId=2025-11-03)。

- 简单地说，就是绳子上有很多气球（字符串表示），每个气球都有拆下来的代价（整型数组），求把绳子上的气球拆得没有连续颜色需要的最小代价。
> https://leetcode.cn/problems/minimum-time-to-make-rope-colorful/solutions/3808739/tan-xin-mei-duan-bao-liu-zui-da-de-pytho-4xln/?envType=daily-question&envId=2025-11-03

# 思路分享
- 贪心策略，先计算代价之和，然后去除每一段颜色中，代价最大的气球即可。

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def minCost(self, colors: str, neededTime: List[int]) -> int:
        ans = cur_max = 0
        n = len(colors)
        for i, x in enumerate(neededTime):
            ans += x
            if cur_max < x:
                cur_max = x
            if i == n-1 or colors[i] != colors[i+1]:
                ans -= cur_max
                cur_max = 0
        return ans
```
