# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/get-equal-substrings-within-budget/)。

- 简单地说，就是给两个字符串，和一个整数变量，求一个最长字符串的长度，使得一个字符串变换为另一个（ASCII码差值），变换的总值在那个整数之内。

# 思路分享
- 典型的不定长滑动窗口（{% post_link 3090每个字符最多出现两次的最长字符串 %}，无需多盐

# 代码解答（强烈建议自行解答后再看）
- 陆爻齐的解法
```c++
class Solution {
public:
    int equalSubstring(string s, string t, int maxCost) {
        int curCost = 0;
        int left = 0;
        int n = t.size();
        int ans = 0;
        for (int i = 0; i < n; i++) {
            curCost += abs(s[i]-t[i]);
            if (curCost > maxCost) {
                curCost -= abs(s[left]-t[left]);
                left++;
            }
            ans = max(ans, i-left+1);
        }
        return ans;
    }
};
```
