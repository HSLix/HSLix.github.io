# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/apple-redistribution-into-boxes/description/?envType=daily-question&envId=2025-12-24)。

- 简单地说，就是给两个一维数组，第一个代表一袋袋苹果，每个元素代表每一袋苹果个数；第二个代表一个个箱子的容量。现在需要把苹果都放箱子里，求需要箱子的最小值。

# 思路分享
- 要求所用最小箱子数，贪心的思想表示，用的箱子大，个数就会小，而且与苹果的袋数没什么关系。这也是灵神的思路，“简单题，简单做”~
> https://leetcode.cn/problems/apple-redistribution-into-boxes/solutions/2678660/pai-xu-tan-xin-pythonjavacgo-by-endlessc-8x64/?envType=daily-question&envId=2025-12-24

- 不过这里也会频繁的取数组最大值，除了排序后处理外，突然很想用堆，毕竟之前天天用。当然也是可以的，在苹果用完前，不断取最大容量箱子使用即可。

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def minimumBoxes(self, apple: List[int], capacity: List[int]) -> int:
        sum_a = sum(apple)
        for i in range(len(capacity)):
            capacity[i] = -capacity[i]
        heapq.heapify(capacity)
        ans = 0
        while sum_a > 0:
            ans += 1
            sum_a += heapq.heappop(capacity)
        return ans
```
