# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/count-partitions-with-even-sum-difference/description/?envType=daily-question&envId=2025-12-05)。

- 简单地说，就是给一个数组，要将这个数组化为左右两部分的非空分区，对这两个分区求和，然后作差，求差值为偶数的分区方案。

# 思路分享
- 这里刚开始的思路是模拟求和做差的过程，所以就会用到前缀和。

- 但是在写的过程中想到，如果全体数和为奇数，那么怎么分，差值也一定为奇数；如果全体数和为偶数，无论新加入左子数组的是奇数还是偶数，两部分的差值都是偶数（差值+2*变化值），所以本质上如果全体数和为偶数，只要给出数组长度减一的值即可。
> https://leetcode.cn/problems/count-partitions-with-even-sum-difference/solutions/3843925/tong-ji-yuan-su-he-chai-zhi-wei-ou-shu-d-mruw/?envType=daily-question&envId=2025-12-05

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def countPartitions(self, nums: List[int]) -> int:
        all_sum = sum(nums)
        if all_sum % 2 != 0:
            return 0
        return len(nums)-1
```
