# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/intersection-of-two-linked-lists/description/)。

- 简单地说，就是给两个链表头，判断这两个链表是否相交。

# 思路分享
- 利用了如果链表相交，那么两个链表走到头后，互换位置再一起走，一定能走到相交位置（如果有的话）
> https://leetcode.cn/problems/intersection-of-two-linked-lists/solutions/2958778/tu-jie-yi-zhang-tu-miao-dong-xiang-jiao-m6tg1/

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        p, q = headA, headB
        while not p == q:
            p = p.next if p else headB
            q = q.next if q else headA
        return p
```
