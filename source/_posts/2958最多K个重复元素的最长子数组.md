# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/length-of-longest-subarray-with-at-most-k-frequency/)。

- 简单地说，就是求一个整数子数组的最长长度，但该子数组的每个元素重复次数不允许超过k个。

# 思路分享
- 不定长滑动窗口（{% post_link 3090每个字符最多出现两次的最长字符串 %}）的经典应用，而且只要把重复的阈值设定为k，那么该题解（{% post_link 1695删除子数组的最大得分 %}）也能用。总之很简单，直接看代码吧。

# 代码解答（强烈建议自行解答后再看）
- 陆爻齐的解法
```c++
class Solution {
public:
    int maxSubarrayLength(vector<int>& nums, int k) {
        int ans = 0;
        int left = 0;
        int n = nums.size();
        unordered_map<int, int> rec;
        for (int i = 0; i < n; i++) {
            int m = nums[i];
            rec[m]++;
            while(rec[m]>k) {
                rec[nums[left++]]--;
            }
            ans = max(ans, i-left+1);
        }
        return ans;
    }
};
```
