# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/max-consecutive-ones-iii/description/)。

- 简单地说，就是给一个由 0 和 1 组成的整数数组，现在最多可以把其中 k 个 0 反转成 1。求反转后最长连续 1 的个数。

# 思路分享
- 纯粹的不定长滑动窗口（{% post_link 3090每个字符最多出现两次的最长字符串 %}），无需多盐。

# 代码解答（强烈建议自行解答后再看）
- 陆爻齐的解法
```c++
class Solution {
public:
    int longestOnes(vector<int>& nums, int k) {
        int zero_count = 0;
        int left = 0;
        int n = nums.size();
        int ans = 0;
        for (int i = 0; i < n;i++) {
            zero_count += 1-nums[i];
            while(zero_count > k) {
                zero_count -= 1-nums[left++];
            }
            ans = max(ans, i-left+1);
        }
        return ans;
    }
};
```
