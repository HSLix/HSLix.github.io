# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/sum-root-to-leaf-numbers/)。

- 简单地说，就是从二叉树根到叶，每个节点的数值从左到右视为一个数字的一位组合，求所有这样数字之和。

# 思路分享
- 从上一层到下一个就把此前的和乘 10 进位（起始 0），然后到叶节点时就检查是否满足题目要求即可。
> https://leetcode.cn/problems/sum-root-to-leaf-numbers/solutions/2730644/jian-ji-xie-fa-pythonjavacgojsrust-by-en-gbu9/

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def sumNumbers(self, root: Optional[TreeNode], cur:int = 0) -> int:
        cur *= 10
        cur += root.val
        if not root.left and not root.right:
            return cur
        return (self.sumNumbers(root.left, cur) if root.left else 0) + (self.sumNumbers(root.right, cur) if root.right else 0)
```
