# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/remove-duplicates-from-sorted-list-ii/)。

- 简单地说，就是给一个链表，对于其中重复的元素，全部删除，只剩没有重复的结点。

# 思路分享
- 与 {% post_link 83删除排序链表中的重复元素 %} 不同的是，这里不会保留重复元素的值，会全删，所以这里需要编辑的是正在遍历结点的下两个结点，因为如果下两个结点值相同，那么当前结点的 next 至少会是后面的第三个结点。
> https://leetcode.cn/problems/remove-duplicates-from-sorted-list-ii/solutions/2004067/ru-he-qu-zhong-yi-ge-shi-pin-jiang-tou-p-2ddn/

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def deleteDuplicates(self, head: Optional[ListNode]) -> Optional[ListNode]:
        internal = ListNode(-101, head)
        cur = internal
        while cur.next and cur.next.next:
            val = cur.next.val
            if cur.next.next.val == val:
                while cur.next and cur.next.val == val:
                    cur.next = cur.next.next
            else:
                cur = cur.next
        return internal.next
```
