# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/maximum-absolute-sum-of-any-subarray/description/)。

- 简单地说，就是要求一个数组内所有子数组中，子数组和的绝对值的最大值。

# 思路分享
- 基本前缀和的思路，可以参考 {% post_link 303区域和检索-数组不可变 %}

- 只要前缀和直接找其中的最大值和最小值求差即可，不过起始值必须是 0，因为得考虑第一个数字本身。

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def maxAbsoluteSum(self, nums: List[int]) -> int:
        s = list(accumulate(nums, initial = 0))
        # print(s)
        return max(s) - min(s)
```
