# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/rotate-list/description/)。

- 简单地说，就是给一个链表，现在要进行 k 次轮换操作，每次轮换会把最后一个结点放到第一个，求最终链表。

# 思路分享
- 这边的参考解法采取了先遍历长度，求得真正的轮换长度（k % 链表长度），然后再借助哨兵结点做指针变换。

- 大部分的其它解法采用把链表变成环（第一次求长度时），然后再往前走真正轮换长度，把环切断变回普通链表得到答案。这种方法更简单一点。
> https://leetcode.cn/problems/rotate-list/solutions/3863642/shou-wei-xiang-lian-cheng-huan-ran-hou-z-412k/

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def rotateRight(self, head: Optional[ListNode], k: int) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        left, right = dummy, dummy

        cur = head
        l = 0
        while cur:
            cur = cur.next
            l += 1
            # if not cur.next:
            #     cur.next = head

        if l == 0:
            return head

        k %= l 
        if k == 0:
            return head

        for _ in range(k):
            if right.next:
                right = right.next
            else:
                right = dummy
        
        while right.next:
            left, right = left.next, right.next

        dummy.next = left.next
        left.next = right.next
        right.next = head

        return dummy.next

```
