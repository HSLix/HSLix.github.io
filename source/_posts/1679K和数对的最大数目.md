# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/max-number-of-k-sum-pairs/)。

- 简单地说，就是找出一个整型数组中有多少对数字和为 K。（不重复）

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def maxOperations(self, nums: List[int], k: int) -> int:
        rec = Counter()
        res = 0
        for num in nums:
            couple = k - num
            #print(couple)
            if rec[couple]:
                res += 1
                rec[couple] -= 1
            else:
                rec[num] += 1
            #print(rec)
        return res
```
