# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/count-collisions-on-a-road/?envType=daily-question&envId=2025-12-04)。

- 简单地说，就是给一个一维数组，其中有三个值，L， S， R， 分别代表向左，静止，向右的车子，如果车碰撞静止车，则碰撞次数加一，如果两个动的车相碰，则碰撞次数加二。求碰撞次数之和。

# 思路分享
- 陆爻齐采用了栈来模拟碰撞情况，实际上有更巧妙的方法，动碰静加一，动碰动加二，说明动车计数一次，那么只要去除左侧向左和右侧向右的车子，剩下会动的必定碰撞计数即可。
> https://leetcode.cn/problems/count-collisions-on-a-road/solutions/1352598/jie-lun-ti-san-xing-gao-ding-by-endlessc-bvnw/?envType=daily-question&envId=2025-12-04

    def countCollisions(self, directions: str) -> int:

        directions = directions.rstrip("R")
        return len(directions) - directions.count("S")
