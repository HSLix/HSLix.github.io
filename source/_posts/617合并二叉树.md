# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/merge-two-binary-trees/description/)。

- 简单地说，就是把两个二叉树合并，都有的节点数值合并，不为 null 的节点直接作为新二叉树的节点。

# 思路分享
- 官方题解镇楼
> https://leetcode.cn/problems/merge-two-binary-trees/solutions/424201/he-bing-er-cha-shu-by-leetcode-solution/

- 这边的思考过程是：1. 不要拘泥于细节，所以关注二叉树本身节点和左右子树；2. 对于两边都 null，那返回 null；3. 对于一边 null，就直接返回另一边，因为剩下的也不会有另一个点，所以不需要再递归；4. 如果两边都有点，那么需要新建点

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def mergeTrees(self, root1: Optional[TreeNode], root2: Optional[TreeNode]) -> Optional[TreeNode]:
        if not root1 and not root2:
            return None 
        if not root2:
            return root1
        if not root1:
            return root2
        return TreeNode(root1.val+root2.val, self.mergeTrees(root1.left, root2.left), self.mergeTrees(root1.right, root2.right))
```
