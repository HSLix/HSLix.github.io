# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/meeting-rooms-iii/description/?envType=daily-question&envId=2025-12-27)。

- 简单地说，就是给一系列会议的开始时间和结束时间，以及会议室个数，现在规定，每个会议都会默认用当前空闲会议室中最小的，如果没有会议室空闲就延期开始，求使用最多的会议室编号。

# 思路分享
- 陆爻齐想到了双堆处理空闲会议室和开始的会议，但由于实现得不好失败了，根据灵神的解法，重点在于处理会议的延期，只有没有空闲的时候才处理比较容易不出错。

- 剩下的就是比较平常的东西，简单过一下，遍历会议，在处理会议开会前，先处理已经结束的会议释放房间，如果没有空闲房间就处理延期，也就是增加当前会议的结束时间。从上面的思路来看，堆只要记录房间号和结束时间即可，到点就放房间。

```python
class Solution:
    def mostBooked(self, n: int, meetings: List[List[int]]) -> int:
        rec = [0]*n
        ans = 0
        max_vis = 0
        room = list(range(n))
        heapq.heapify(room)
        meetings.sort(key=lambda x : x[0])
        on_meetings = []
        heapq.heapify(on_meetings)
        # print(meetings)

        for start, end in meetings:
            while on_meetings and on_meetings[0][0] <= start:
                heappush(room, heappop(on_meetings)[1])
            e = 0
            if len(room) == 0:
                e, r = heapq.heappop(on_meetings)
                heapq.heappush(room, r)
                end += e - start
            nxt_room = heapq.heappop(room)
            heapq.heappush(on_meetings, (end, nxt_room))
            rec[nxt_room] += 1

        return rec.index(max(rec))
```
