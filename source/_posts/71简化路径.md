# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/simplify-path/description/)。

- 简单地说，就是给一个字符串表示Unix的文件路径，其中可能用 . 或 .. 表示本级目录和上级目录，以及多余的 /，需要处理。最终给出最简的路径样子。

# 思路分享
- 本题的思路比较简单，只要用栈模拟路径内容的从左往右输入即可，但是中间的具体实现方式得学学 Python 相关的函数。

- 比如，本题分割字符串，就用了 split() 函数，该函数能根据分割符来分割内容。还有 join 函数能够根据给的符号把整个 list 给结合起来。有了这两个方法就能方便地实现本题的需求。
> https://leetcode.cn/problems/simplify-path/solutions/3042801/zhan-mo-ni-pythonjavacgojsrust-by-endles-y4bc/

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def simplifyPath(self, path: str) -> str:
        stack = list()
        
        for s in path.split('/'):
            if s == "" or s == '.':
                continue
            if s != "..":
                stack.append(s)
            elif stack:
                stack.pop()
        
        return '/' + '/'.join(stack)
```
