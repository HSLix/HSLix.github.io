# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/maximum-matrix-sum/description/?envType=daily-question&envId=2026-01-05)。

- 简单地说，就是给一个方阵，现在可以对其中做任意次的操作，即将相邻的两个元素乘以 -1，求操作后方阵的最大和是多少。

# 思路分享
- 膜拜灵神，将这道题分解，发现经过任意次操作，可以将两个数字乘 -1.

- 也就是说：如果有偶数个负数就能全部变正，如果只有奇数个的情况下，如果有 0，那也可以全变成正数，否则就要选取其中数值最小的负数。
> https://leetcode.cn/problems/maximum-matrix-sum/solutions/950421/go-tan-xin-by-endlesscheng-opwm/?envType=daily-question&envId=2026-01-05

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def maxMatrixSum(self, matrix: List[List[int]]) -> int:
        total, negative_count = 0, 0
        min_negative = inf
        for row in matrix:
            for x in row:
                if x < 0:
                    x = -x
                    negative_count += 1
                min_negative = min(min_negative, x)
                total += x
        if negative_count % 2:
            total -= min_negative * 2 # 乘二是因为这个数之间加上去过，要先去除之前的值再减
        return total
```
