# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/maximum-length-substring-with-two-occurrences/)。

- 简单地说，就是给一个字符串，求里面子串的最长长度，该子串需要每个字母不超过两个。

# 思路分享
- 不定长滑动窗口，在这里正式引入灵神的模板，也就是循环扩大窗口右侧，当条件不满足时，让窗口左侧右移，直到满足条件。
> https://leetcode.cn/problems/maximum-length-substring-with-two-occurrences/solutions/2704776/on-hua-dong-chuang-kou-pythonjavacgo-by-hl44d/

- 简单地把2改成1就能用在上一题上力，就是 {% post_link 3无重复字符的最长子串 %}。

# 代码解答（强烈建议自行解答后再看）
- 参考灵神的模板答案
```c++
class Solution {
public:
    int maximumLengthSubstring(string s) {
        vector<int> rec(26, 0);
        int left = 0;
        int n = s.size();
        int ans = 0;
        for (int i = 0; i < n; i++) {
            int index = s[i]-'a';
            rec[index]++;
            while (rec[index] > 2) {
                rec[s[left++]-'a']--;
            }
            ans = max(ans, i-left+1);
        }
        return ans;
    }
};
```
