# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def countTriples(self, n: int) -> int:
        rec = [0] * (250**2+1)
        ans = 0
        for i in range(1, n+1):
            rec[i**2] += 1
            for j in range(1, i):
                ans += rec[i**2-j**2]
        return ans
```
