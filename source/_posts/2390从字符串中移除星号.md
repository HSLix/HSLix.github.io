# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/removing-stars-from-a-string/description/)。

- 简单地说，就是把字符串中的星号当作退格，每次遇到就删除前面一个字符，求最终处理后的字符串。

# 思路分享

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def removeStars(self, s: str) -> str:
        rec = list()
        for c in s:
            if c != '*':
                rec.append(c)
            elif rec:
                rec.pop()

        return "".join(rec)
```
