# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/count-nodes-equal-to-average-of-subtree/description/)。

- 简单地说，就是给一个二叉树，求其中子树的平均值与子树根节点值相等的个数。

# 思路分享
- 外部记录符合条件的个数，内部 dfs 传递平均值和节点数用于计算平均值即可。
> https://leetcode.cn/problems/count-nodes-equal-to-average-of-subtree/solutions/1477315/tong-ji-zi-shu-de-jie-dian-he-ji-jie-dia-va8t/

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def averageOfSubtree(self, root: TreeNode) -> int:
        ans = 0
        def dfs(root:TreeNode):
            if not root:
                return 0, 0
            left_sum, left_count = dfs(root.left)
            right_sum, right_count = dfs(root.right)
            cur_sum = left_sum + right_sum + root.val
            cur_count = left_count + right_count + 1
            if root.val == (cur_sum // cur_count):
                nonlocal ans 
                ans += 1
            return cur_sum, cur_count
        dfs(root)
        return ans
```
