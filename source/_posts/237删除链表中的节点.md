# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/delete-node-in-a-linked-list/description/)。

- 简单地说，就是在不给链表头的情况下，仅给要删除的节点，如何实现删除。（链表长度至少为 2，且删除节点不在队尾）

# 思路分享
- 正如灵神所说，真是脑筋急转弯，由于检测是否删除靠得是输出值，所以只要把要删除节点的值换成下一个节点，并把 next 跳过下一个就行。
> https://leetcode.cn/problems/delete-node-in-a-linked-list/solutions/2004056/ru-he-shan-chu-jie-dian-liu-fen-zhong-ga-x3kn/

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def deleteNode(self, node):
        """
        :type node: ListNode
        :rtype: void Do not return anything, modify node in-place instead.
        """
        node.val = node.next.val
        node.next = node.next.next
```
