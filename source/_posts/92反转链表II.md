# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/reverse-linked-list-ii/description/)。

- 简单地说，就是要将链表中的一段反转，这一段可以是任意长度任意位置的一段。

# 思路分享
- 与 {% post_link 206反转链表 %} 有区别，不是全部反转，所以除了反转内部分的处理，重点是反转前和反转后节点的接入处理。

- 反转内的就不赘述了，这里的新东西是，需要找到反转段的前一个节点，设为 p0，由于反转后产生的一个变量正好有反转后的第一个节点，设为 p1，这个节点还连着原来的下一个，正好就让 p0 现在连着的下一个（反转后的末尾）连 p1 的下一个，然后让 p 0 连 p1 就好了。
> https://leetcode.cn/problems/reverse-linked-list-ii/solutions/1992226/you-xie-cuo-liao-yi-ge-shi-pin-jiang-tou-teqq/

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseBetween(self, head: Optional[ListNode], left: int, right: int) -> Optional[ListNode]:
        dummy = ListNode(-1, head)
        p0 = dummy
        for _ in range(left-1):
            p0 = p0.next
        
        pre = None
        cur = p0.next
        for _ in range(right-left+1):
            nxt = cur.next
            cur.next = pre
            pre = cur
            cur = nxt 
        
        p0.next.next = cur
        p0.next = pre

        return dummy.next
```
