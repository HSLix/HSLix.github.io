# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/four-divisors/description/?envType=daily-question&envId=2026-01-04)。

- 简单地说，就是一堆数字，从中找到恰好四个因数的，把求恰四因数的因数和。

# 思路分享
- 采用预处理，空间换时间，直接记录范围内所有数字的因数数量以及因数和，然后 O(1) 计算。
> https://leetcode.cn/problems/four-divisors/solutions/3867962/mo-ban-yu-chu-li-mei-ge-shu-de-yin-zi-ge-e0v6/?envType=daily-question&envId=2026-01-04

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
MX = 100_001
divisor_num = [0]*MX
divisor_sum = [0]*MX
for i in range(1, MX):
    for j in range(i, MX, i):
        divisor_num[j] += 1
        divisor_sum[j] += i  

class Solution:
    def sumFourDivisors(self, nums: List[int]) -> int:
        ans = 0
        for num in nums:
            if divisor_num[num] == 4:
                ans += divisor_sum[num]
        return ans
```
