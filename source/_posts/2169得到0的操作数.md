# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/count-operations-to-obtain-zero/description/?envType=daily-question&envId=2025-11-09)。

- 简单地说，就是给两个数字，规定只能让大的数字减去小的数字，直到其中有一个数字归零，求这个操作数。

# 思路分享
- 这是个简单题，所以直接两个数字模拟这个大减小计数就行。重点在于找到其本质，来加深相关知识。
> https://leetcode.cn/problems/count-operations-to-obtain-zero/solutions/1262427/olog-zhan-zhuan-xiang-chu-fa-by-endlessc-vxq0/?envType=daily-question&envId=2025-11-09

- 对于本题，不断减去大的数字，直到该数字比另一个数字少，这个过程本质是求余数，而把余数和除数倒转的样子，不就是辗转相除法嘛（GCD），而本题所求的操作数，本质上就是 GCD 过程中产生的除数之和。

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def countOperations(self, num1: int, num2: int) -> int:
        ans = 0
        while num2 > 0:
            ans += num1 // num2
            num1, num2 = num2, num1 % num2
        return ans
```
