# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/largest-positive-integer-that-exists-with-its-negative/description/)。

- 简单地说，就是给一个数组，找一种一组值相加为 0，且在多组这样的值中，选出正数最大的数值返回。

- 参考题解
```c++
class Solution {
public:
    int findMaxK(vector<int>& nums) {
        int max_int = -1;
        unordered_set<int> rec;
        for (const int &i : nums) {
            if (rec.count(-i)) {
                max_int = max(max_int, abs(i));
            }
            rec.insert(i);
        }
        return max_int;
    }
};
```
