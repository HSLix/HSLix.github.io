# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def minEatingSpeed(self, piles: List[int], h: int) -> int:
        left, right = 0, max(piles)+1
        while left+1 < right:
            mid = (left+right) // 2

            # 这一步是验证答案
            use = sum((num-1)//mid + 1 for num in piles)

            if use <= h:
                right = mid
            else:
                left = mid

        return right
```
