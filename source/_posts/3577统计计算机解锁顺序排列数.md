# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/count-the-number-of-computer-unlocking-permutations/?envType=daily-question&envId=2025-12-10)。

- 简单地说，就是给一个数组，现在可以用数组的开始的一个的数字作为起点（这个是定值，不是位置），对后续比该数字大的进行“解锁”，求将所有数字解锁的排列方案。

# 思路分享
- 由于只能小解锁大，所以如果存在与该值同值或者更小的值就是非法，除此之外，其它数字的排列直接An-1，也就是 (n-1)!。

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def countPermutations(self, complexity: List[int]) -> int:
        root = complexity[0]
        complexity.sort()
        MOD = 1_000_000_007
        # print(root, complexity)
        # print(root > complexity[0])
        # print(root == complexity[1])
        if root > complexity[0] or root == complexity[1]:
            return 0
        ans = 1
        for i in range(1, len(complexity)):
            # print(ans)
            ans = (ans * i) % MOD
        return ans
```
