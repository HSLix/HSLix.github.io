# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/binary-tree-right-side-view/description/)。

- 简单地说，就是给一个二叉树，然后返回二叉树每层最右边的节点列表。

# 思路分享
- 灵神的思路很牛逼，采用递归的方法，每次递归检测当前递归深度是否达到答案的长度，如果是，说明这是本层第一次遍历，然后只要保证每层先往右遍历就能实现第一个就是最右边。
> https://leetcode.cn/problems/binary-tree-right-side-view/solutions/2015061/ru-he-ling-huo-yun-yong-di-gui-lai-kan-s-r1nc/

- 不过灵神的方法没那么直观，参考题解采用层序遍历，直观地一层层节点，取第一个（也是保证先把右边的放进遍历列表）

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def rightSideView(self, root: Optional[TreeNode]) -> List[int]:
        if not root:
            return []
        ans = []
        visit = deque([root])
        while visit:
            ans.append(visit[0].val)
            cnt_vis = len(visit)
            for _ in range(cnt_vis):
                cur = visit.popleft()
                if cur.right:
                    visit.append(cur.right)
                if cur.left:
                    visit.append(cur.left)
        return ans
```
