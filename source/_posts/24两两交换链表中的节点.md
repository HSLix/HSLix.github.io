# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/swap-nodes-in-pairs/)。

- 简单地说，对于链表中的内容，就两两交换，这里的两两交换不重合。

# 思路分享
- 对于一个链表中的两个节点互换只要想清楚（画清楚）就不难，本题只要注意好边界条件就好做，一个是开始的 0 个、 1 个节点，二个是奇数个节点，这里在循环条件设置了检测只有两个节点的位置才做。
> https://leetcode.cn/problems/swap-nodes-in-pairs/solutions/2374872/tu-jie-die-dai-di-gui-yi-zhang-tu-miao-d-51ap/

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def swapPairs(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head or not head.next:
            return head
        
        dummy = ListNode(-1, head)
        pre = dummy
        while pre.next and pre.next.next:
            cur = pre.next
            last = cur.next
            nxt = last.next
            last.next = cur
            pre.next = last
            cur.next = nxt
            pre = cur
        return dummy.next

```
