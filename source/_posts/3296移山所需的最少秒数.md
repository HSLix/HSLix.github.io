# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/minimum-number-of-seconds-to-make-mountain-height-zero/description/)。

- 简单地说，就是给一个数字代表山的高度，和一个数组代表工人挖一单位山的工作时间，工人每挖一次，下次会耗时更久，求工人们挖空山的最短时间。

# 思路分享
- 用最小堆模拟工人的挖山时间变化即可，不过灵神的解法似乎利用了局部变量的泄露，还挺巧妙的，保险起见还是在外面声明一下好点。
> https://leetcode.cn/problems/minimum-number-of-seconds-to-make-mountain-height-zero/solutions/2925848/er-fen-da-an-pythonjavacgo-by-endlessche-myg4/

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def minNumberOfSeconds(self, mountainHeight: int, workerTimes: List[int]) -> int:
        worker = [(t, t, t) for t in workerTimes]
        heapq.heapify(worker)
        nxt = 0
        for _ in range(mountainHeight):
            nxt, delta, base = worker[0]
            heapq.heapreplace(worker, (nxt+delta+base, delta+base, base))
        return nxt
```
