# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/odd-even-linked-list/description/)。

- 简单地说，就是给一个奇数和偶数轮流的链表，现在需要把这个链表重组，使得奇数都放前面，偶数放奇数后面。

# 思路分享
- 官方题解镇楼
> https://leetcode.cn/problems/odd-even-linked-list/solutions/482737/qi-ou-lian-biao-by-leetcode-solution/

- 参考题解和官方的差不多，都是发现奇偶只要各用一个指针管理，由于奇偶交叉，所以让偶数指针指向奇数指针的下一个，让奇数指针指向偶数指针原来的下一个，以此类推即可。

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def oddEvenList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head or not head.next:
            return head

        even_head = head.next
        odd_cur = head
        even_cur = head.next

        while True:
            if not odd_cur.next or not even_cur:
                break
            
            odd_cur.next = even_cur.next
            if odd_cur.next:
                odd_cur = odd_cur.next

            even_cur.next = odd_cur.next
            even_cur = even_cur.next

        odd_cur.next = even_head
        return head
```
