# 题目浅析
- 想查看原题可以点击[题目链接](https://leetcode.cn/problems/unique-paths/description/)。

- 简单地说，就是给一个 m*n 的网格，求从该网格的左上角到右下角有多少条路径。

# 思路分享
- 先划分子问题，一个不错的思路是考虑最后一步，（m-1，n-1）处是由（m-1，n-2）和（m-2，n-1）相加得出的，这样就算得到了子问题和状态转移方程，接下来转记忆化搜索（本题比较容易就直接递推了）再翻译成递推，进而优化空间。
>https://leetcode.cn/problems/unique-paths/solutions/3062432/liang-chong-fang-fa-dong-tai-gui-hua-zu-o5k32/

# 代码解答（强烈建议自行解答后再看）
- 参考题解
```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        f = [1]*n
        for i in range(1, m):
            for j in range(1, n):
                f[j] = f[j] + f[j-1]
        return f[n-1]
```
